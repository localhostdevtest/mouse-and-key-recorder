Documentación de API para Postman

Todos los endpoints disponibles para usar desde Postman o cualquier cliente HTTP

Navegación
Uso Básico Automatizaciones Secuencias Grabación Ejemplos

🌐 Información Básica

Base URL: http://localhost:5000

Formato: Todas las respuestas son JSON

Headers requeridos: Content-Type: application/json (solo para POST/PUT)

🎯 Automatizaciones

Listar todas las automatizaciones
GET /api/tasks/list
Ejecutar UNA automatización
POST /api/run/NOMBRE_AUTOMATIZACION

Ejemplo: POST /api/run/mi_automatizacion

Eliminar automatización
DELETE /api/tasks/delete/NOMBRE_AUTOMATIZACION

🔄 Secuencias

Ejecutar VARIAS automatizaciones al instante
POST /api/quick-sequence

Body JSON:

{
  "tasks": ["automatizacion1", "automatizacion2", "automatizacion3"],
  "pause_between": 2.0
}
Crear secuencia permanente
POST /api/sequences/create

Body JSON:

{
  "name": "Mi_Secuencia",
  "tasks": ["automatizacion1", "automatizacion2"],
  "description": "Descripción opcional",
  "pause_between": 1.5
}
Ejecutar secuencia guardada
POST /api/run-sequence/NOMBRE_SECUENCIA
Listar secuencias
GET /api/sequences/list

🔴 Control de Grabación

Iniciar grabación
POST /api/recording/start
Detener grabación
POST /api/recording/stop
Ver estado de grabación
GET /api/recording/status
Guardar grabación como automatización
POST /api/tasks/save

Body JSON:

{
  "name": "Mi_Automatizacion",
  "description": "Descripción opcional"
}

🚀 Ejemplos Prácticos

Ver todo lo que tienes
GET http://localhost:5000/api/list-all
Ejecutar una automatización desde curl
curl -X POST http://localhost:5000/api/run/mi_automatizacion
Ejecutar 3 automatizaciones en secuencia desde curl
curl -X POST http://localhost:5000/api/quick-sequence \
-H "Content-Type: application/json" \
-d '{"tasks": ["auto1", "auto2", "auto3"], "pause_between": 1.0}'
Tip

Puedes usar estos endpoints desde Postman, curl, Python requests, o cualquier cliente HTTP. Todos devuelven JSON con {"success": true/false, "message": "..."}
